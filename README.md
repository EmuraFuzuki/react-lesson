# react-lesson

これはReactの基本事項を学ぶためのリポジトリです。Reactを利用したwebアプリケーションの開発に必要な知識を段階的に解説します。

## 目次

### 1. Reactによるwebアプリケーションの基礎

カウンターアプリの作成を通して、Reactの基本的な概念を学びます。

- ReactとHTML
- TypescriptとjavaScript
- プロジェクトの作成
- コンポーネント
- Hooks

## 2. 応用

テーマカラーの導入を通して、Reactの応用的な技術を学びます。

- 型システム
- カスタムフックによるカプセル化
- 状態管理 (jotai - atom)
- tailwindcss

## 3. 設計技法

電卓を作成して、設計技法を学びます。

- SOLID原則
- ドメイン駆動設計

## 4. React / TypeScript を理解する

ReactとTypescriptの動作原理に踏み込んだ説明をします。

- Reactのレンダリングの仕組み
- useStateとuseRef
- useEffect
- typescriptの型システム
- クロージャー
